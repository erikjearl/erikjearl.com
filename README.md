# erikjearl.com

Personal website for Erik Earl — software engineer, climber, hiker.

## Status

Design phase complete; implementation not started.

- **Approved design reference:** `mockups/d-immersive.html` — open it in a browser. This mockup *is* the approved first draft of the site.
- **Design docs:** `docs/superpowers/specs/2026-07-10-visual-design.md` (visual spec + decision log) and `2026-07-10-architecture-design.md` (React + Vite + TS static build, GitHub Pages, deferred Flask phase 2).
- `mockups/a-ascent.html`, `b-topo.html`, `c-minimal.html` — earlier concept explorations, kept for reference.
- `mockups/assets/` — web-converted photos used by the mockups (originals live in `media/`, untracked).

## Concept in one line

Scroll-as-ascent: one continuous page that travels from the Yosemite valley floor in daylight to a desert night sky — the climbing identity carried entirely by Erik's own photography, color, and motion, while the written content stays strictly professional.
